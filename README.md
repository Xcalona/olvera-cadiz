## 🚀 Cómo Usar

### Opción 1: Ver Localmente

1. Clona el repositorio:
```bash
git clone https://github.com/Xcalona/olvera-cadiz.git
cd olvera-cadiz
```

2. Abre el archivo `index.html` en tu navegador:
```bash
open index.html
# O en Windows:
start index.html
```

### Opción 2: GitHub Pages

1. Ve a **Settings** → **Pages**
2. Selecciona `main` como rama fuente
3. Espera a que se deployed
4. Tu sitio estará disponible en: `https://xcalona.github.io/olvera-cadiz`

## 🔧 Personalización

### Cambiar Colores
Edita las variables en `styles.css`:
```css
:root {
    --primary-color: #d4a574;
    --secondary-color: #2c3e50;
    --accent-color: #e74c3c;
}
```

### Agregar Contenido
1. Edita `index.html` para cambiar textos y secciones
2. Mantén la estructura de clases para que los estilos se apliquen correctamente

### Integrar Imágenes Reales
Reemplaza los emojis (🏰, ⛪, etc.) con URLs de imágenes:
```html
<!-- En lugar de: -->
<div class="gallery-image">🏰</div>

<!-- Usa: -->
<img src="url-imagen.jpg" alt="Castillo de Olvera" class="gallery-image">
```
